## 一、需求文档

### 1.1 项目背景

海洋生物多样性是地球生态系统的重要组成部分。本系统旨在为广东海洋大学科研人员、学生及公众提供一个统一的海洋生物信息管理与共享平台，实现对海洋生物物种数据、生态系统信息及实地观测记录的数字化管理。

### 1.2 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 管理员 | 系统管理人员 | 全部操作权限（增删改查 + 用户管理） |
| 科研人员 | 高校研究人员 | 增删改查物种、观测记录、生态系统 |
| 学生 | 在校学生 | 查询、添加观测记录 |
| 访客/公众 | 普通用户 | 只读查询 |

### 1.3 功能需求清单

#### 模块一：用户权限管理
- FR-01：用户注册（邮箱/用户名+密码）
- FR-02：用户登录 / 退出（JWT Token 认证）
- FR-03：基于 RBAC 的角色权限控制
- FR-04：个人资料查看与修改
- FR-05：管理员对用户进行增删改查与角色分配
- FR-06：操作日志记录

#### 模块二：物种信息管理（核心）
- FR-07：新增物种（中文名、学名、门/纲/目/科/属/种、形态描述、生态习性、地理分布、保护等级、濒危状态、图片、参考文献）
- FR-08：编辑/删除物种信息（管理员/科研人员）
- FR-09：多条件检索（名称模糊搜索、门纲筛选、保护等级筛选、濒危状态筛选）
- FR-10：物种详情展示（含图片、完整分类学信息）
- FR-11：卡片视图与表格视图切换

#### 模块三：生态系统与观测记录管理（核心）
- FR-12：生态系统管理（珊瑚礁、红树林、海草床、深海等分类管理）
- FR-13：新增观测记录（时间、地点、经纬度、天气、水温、盐度、能见度、深度、观测描述）
- FR-14：观测记录关联物种（多对多关系）
- FR-15：观测记录编辑/删除
- FR-16：按时间、地点、物种筛选观测记录

#### 模块四：可视化与统计分析
- FR-17：物种地理分布地图（Leaflet 标记点）
- FR-18：观测地点地图展示
- FR-19：物种门类分布饼图（ECharts）
- FR-20：保护等级统计柱状图
- FR-21：月度观测趋势折线图
- FR-22：综合数据看板（总物种数、总观测次数、生态系统数量等）
- FR-23：数据导出（CSV 格式）

### 1.4 非功能需求

| 类型 | 要求 |
|------|------|
| 性能 | 页面加载时间 < 2秒，列表查询响应 < 500ms |
| 安全 | JWT 认证、RBAC 权限控制、前端路由守卫 |
| 可用性 | 系统可用率 ≥ 99%，响应式布局 |
| 可维护性 | 组件化开发，代码注释覆盖率 ≥ 60% |
| 测试 | 单元测试覆盖率 ≥ 80%，核心用例 100% |
