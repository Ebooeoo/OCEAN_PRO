❓ 常见问题

❌ 前端白屏 / 刷新后 404
已在 netlify.toml 配置了重定向规则（/* → /index.html 200）。如果仍然 404，检查 netlify.toml 是否在 dist 目录里。也可以在 Netlify → Site configuration → Redirects 手动添加 /* /index.html 200。
❌ 接口报错 CORS 跨域
后端 app.js 已设置 cors({ origin: '*' })，正常情况不会报 CORS 错误。如果仍然报错，检查 VITE_API_BASE_URL 是否填写正确，末尾不要加 /。
❌ 后端数据库连接失败
检查 Render 环境变量中的 DB_HOST / DB_PORT / DB_USER / DB_PASSWORD / DB_NAME 是否与 Render MySQL 数据库面板显示的一致。DB_NAME 填 marine_biodiversity。
❌ Render 免费版后端 "冷启动" 很慢
Render 免费版 15 分钟无请求后会休眠，下次访问需要 30-60 秒冷启动。这对作业演示是正常现象。可以提前打开 /api/health 接口"唤醒"后端。
❓ Render MySQL 免费版有什么限制
Render MySQL 免费版有 90 天有效期，到期后数据库会被删除（但可以重新导入）。存储限制 1GB，对于本项目完全够用。
❓ 如何把 Gitee 仓库同步到 GitHub（用于 Netlify Git 连接）
1. 在 GitHub.com 新建一个空仓库（不要初始化）
2. 在本地运行：
git remote add github https://github.com/你的用户名/仓库名.git
git push github master --force
3. 然后在 Netlify 连接 GitHub 仓库即可
